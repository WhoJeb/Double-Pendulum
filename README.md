# Double Pendulum

## Interesting Values
Just a general rule of thumb the smaller the value the easier it is to get a good looking simulation.

Also feels worth noting that these are the initial values (Angular Vel & Position).

---

### 1. Dittos - Doesn't follow a clear pattern
#### 1. 1
- Outer Arm Vel -> 0.2
- Inner Arm Vel -> 0.5
- Outer Arm Pos -> PI/2
- Inner Arm Pos -> PI/2

####  2. 2
- Outer Arm Vel -> 0.2
- Inner Arm Vel -> 0
- Outer Arm Pos -> PI/2
- Inner Arm Pos -> PI/2

#### 3. 3
- Outer Arm Vel -> 0.5
- Inner Arm Vel -> 0
- Outer Arm Pos -> PI/2
- Inner Arm Pos -> PI/2

#### 4. 4
